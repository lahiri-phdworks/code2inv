## Invariants for RQ2 under constrained grammar

```
Binary Search
#define PHI                                                                    \
  ((0 <= low <= mid <= high) &&                                                \
   (((0 <= mid) && (mid < low) && (arr[mid] != item)) ||                       \
    ((high <= mid) && (mid <= SIZE) && (arr[mid] != item)) ||                  \
    (arr[mid] == item))) // generated by code2inv
```

```
Fast Exponentiation
#define PHI                                                                    \
  (a >= 1) && (b >= 0) &&                                                      \
      (res * pow(a, y) == pow(x, y)) // generated by code2inv
```

```
Factorial modulo a constant number (prime)
#define PHI                                                                    \
  (n >= 0) && (k >= 0) && (r == factmod(k, 100003)) // generated by code2inv
```

```
Iterated Fibonacci
#define PHI                                                                    \
  ((0 <= i <= n) && (x == fib(i) && y == fib(i + 1))) // generated by code2inv
```

```
GCD using loop checked against GCD assembly language code.
#define PHI                                                                    \
  (a >= 0) && (b >= 0) &&                                                      \
      (asmgcd(a, b) == asmgcd(x, y)) // generated by code2inv
```

```
GCD using loop.
#define PHI                                                                    \
  (a >= 0) && (b >= 0) && (gcd(a, b) == gcd(x, y)) // generated by code2inv
```

```
Integer Division
#define PHI ((a >= 0) && (d > 0) && (a == q * d + r)) // generated by code2inv
```

```
LCM using loop, checked using GCD formula (GCD computation in assembly code)
#define PHI                                                                    \
  (a > 0) && (b > 0) && (start > 0) &&                                         \
      ((start % a) == 0) // generated by code2inv
```

```
Partition Routine from QuickSort
#define PHI ((i >= 0) && (i < high) && (arr[i - 1] <= pivot))
```

```
Integer Square Root
#define PHI ((n >= 0) && (pow(left - 1, 2) <= n)) // generated by code2inv
```
